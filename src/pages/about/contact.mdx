Since Plxtra is an open source project, it is not possible to directly contact a person.  However you can:
* Start a [discussion](https://github.com/orgs/plxtra/discussions) at [GitHub](https://github.com/plxtra)
* Raise an issue in the appropriate repository at [GitHub](https://github.com/plxtra)
* Contact an organisation which provides [commercial support](/solutions/commercial-support/)
