Plxtra software source code is licensed under the [MIT](https://opensource.org/license/mit) open source license.

This is a [permissive license](https://en.wikipedia.org/wiki/Permissive_software_license) with few restrictions and high [license compatibility](https://en.wikipedia.org/wiki/License_compatibility).  It allows the source code to be used in commercial/proprietary software provided it includes a copy of Plxtra MIT license and also a copyright notice.
